# City Management Platform

A web-based platform to manage and store essential city information including public services, infrastructure, and amenities.

## Features

### Admin Panel
- Add new city services, infrastructure, and amenities
- Manage existing city information
- Monitor infrastructure data
- View reports and feedback statistics

### User Panel
- Search and browse city services
- Report issues with public services
- Provide feedback and ratings for amenities
- Access detailed information about city resources

## Usage

1. Open `index.html` in a web browser
2. Switch between Admin and User modes using the navigation buttons
3. Data is stored locally in browser localStorage

### Admin Functions
- Click "Admin" to access admin panel
- Use "Add City Info" to add new services
- "Manage Services" shows statistics
- "Infrastructure" displays infrastructure items

### User Functions
- Search for services using the search bar
- Click "Report Issue" to report problems
- Click "Give Feedback" to rate services
- Browse all available services in the grid view
